《Spring In Action》学习笔记（三）：Spring 4.0 模块

Spring 4.0含有20个独立的模块（modules），每个模块含有三个文件：

  • binary class file
  • source JAR file
  • JavaDoc JAR file

这20个独立模块可以分为6大类。

Data Access & Integration

  1. JDBC
  2. Transaction
  3. ORM
  4. OXM
  5. Messaging
  6. JMS

Web and Remoting

  1. Web
  2. Web servlet
  3. Web portlet
  4. WebSocket

Aspect-Oriented Programing

  1. AOP
  2. Aspects

Instrumentation

  1. Instrument
  2. Instrument Tomcat

Core Spring Container

  1. Beans
  2. Core
  3. Context
  4. Expression
  5. Context support

Testing

  1. test

附录：缩写

  • DAO：Data-access object
  • ORM：Object-relational mapping
  • MVC：Model-View-Controller
